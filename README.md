# Simple Port Scanner

A lightweight and easy-to-use port scanner for scanning open TCP ports on a target host or range of hosts. Designed for learning, basic network troubleshooting, and security testing.

---

## Features

- **Target Specification:** Scan a single IP, hostname, or a list of targets.
- **Port Selection:** Scan individual ports, port ranges, or common ports.
- **Basic TCP Connect Scanning:** Attempts to establish a TCP connection to determine if a port is open.
- **Result Reporting:** Clearly reports open, closed, or filtered ports.
- **Timeout and Performance Controls:** Configure scan timeouts and concurrency (scan speed).
- **Command-Line Interface:** Simple, flexible CLI with clear options and help.
- **Error Handling:** Gracefully handles invalid input and network errors.
- **(Optional) Banner Grabbing:** Attempts to retrieve basic service banners from open ports.
- **Extensible:** Easily add new features or scan types as needed.

---
## Disclaimer

This tool is intended for educational and authorized testing purposes only. **Always obtain proper authorization before scanning any network or host you do not own.**
